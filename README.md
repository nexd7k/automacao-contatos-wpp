# automacao-contatos-wpp
Código Python para automatizar mensagens usando as bibliotecas webbrowser e pyautogui

Para utilizar, o usuário irá precisar preencher a planilha clientes.xlsx com os dados do contato do whatsapp que queira mandar mensagem. 

Essa automação irá ser executada em seu Google Chrome.

A automação sempre irá ser executada no monitor principal e irá controlar seu mouse/teclado. Portanto, é recomendável executar quando precisar fazer alguma outra coisa fora do dispositivo.
